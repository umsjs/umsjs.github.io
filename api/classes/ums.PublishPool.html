error

错误事件及其事件监听处理程序. 如果添加'error'事件监听函数，则当出现内部错误时回调函数将被调用。

Event Payload:

  • error_msg Error

    错误信息

});

publish

Defined in ums.js:1787

直播流开始发布事件及其事件监听处理程序. 如果添加'publish'事件监听函数，则当一个新的直播流开始发布时回 调函数将被调用。

Event Payload:

  • pipe ums.PublishPipe

    直播流发布管道

Example:

publishPool.on('publish', function(pipe){
    console.log(pipe.name, 'is now publishing...');
});

ready

Defined in ums.js:1776

发布池就绪. 如果添加'ready'事件监听函数，则当发布池就绪时回调函数将被调用。
