enumConnections

  • callback
)
public

Defined in ums.js:3212

枚举向外建立的所有连接对象. 在处理大量连接时，使用该方法比较高效，如果要查找某一符合条件的连接对象， 建议使用该方法。注意：当connection为null时，表明连接集合为空。

Parameters:

  • callback Function

    枚举结果回调函数 callback param conn {ums.RTSPConnection} RTSPConnection isEnd {Boolean} 是否是最后一个枚举值 callback return : isContinue {Boolean} 指示枚举是否继续

Example:

connector.enumConnections(function callback(conn, isEnd){
   var isContinue = true;
   console.log("isEmpty :", !conn, ";isEnd :", isEnd, ";index :", nCount++);
   nCount++;
   if (isEnd) {
       console.log('Count of connections :', nCount);
   }
   return isContinue;
});

    console.dir(err);
});
