createPublishPool

(
  • is_use_default
  • poolOpt
)
public

Defined in ums.js:3504

创建一个ums.PublishPool实例.

Parameters:

  • is_use_default Boolean

    是否使用缺省的实例，而不创建新的PublishPool实例

  • poolOpt Object

    参数集对象，包含级联配置信息，如下 poolOpt.appName {String} 应用名称; poolOpt.rtmpAddress {Object} 含有host或port属性的对象，要注册到redis中的本机对外提供直播服务的主机地址和端口; poolOpt.subEx {String} 为避免混淆而在以流名称为事件名时所添加的前缀; poolOpt.streamEx {String} 为避免混淆而在以流名称为索引查询redis时所添加的前缀; poolOpt.redisAddress {Object} 含有host和port属性的对象，用于指定redis服务器; poolOpt.cluster {Boolean} 指定是否开启自动级联相关接口及功能，当poolOpt.cluster 为false时，PublishPool.publish和PublishPool.subscribe 接口仅能在本地发布/订阅直播流。

Example:

var poolOpt = {
    cluster : true,
    appName : 'live',
    rtmpAddress : {host : '127.0.0.1', port : 1935},
    subEx : '__sub_ex__',
    streamEx : '__livestream_ex__',
    redisAddress : {host:'127.0.0.1', port:6379}
};
// 创建一个直播流发布池
var publishPool = ums.createPublishPool(false, poolOpt);
